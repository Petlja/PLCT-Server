# PLCT Front App

React front-end for PLCT Server, built with [Vite](https://vitejs.dev/).

## Available Scripts

### `npm start` / `npm run dev`

Runs the app in development mode at [http://localhost:3000/app](http://localhost:3000/app).
The page will hot-reload as you make edits.

### `npm run build`

Type-checks with TypeScript, then builds the app for production to the `build` folder and copies it to `../plct_server/front-app/build`.

### `npm run preview`

Locally previews the production build.

### `npm test`

Runs the test suite once using [Vitest](https://vitest.dev/).

### `npm run test:watch`

Runs tests in watch mode.
